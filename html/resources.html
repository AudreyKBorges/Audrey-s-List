Resources

theCoderSchool

3555 Rayford Road, Ste #45 Spring, TX 77386

theCoderSchool teaches coding to kids ages 7-18 of all abilities and skill levels. Kids learn to think like a computer programmer through project-based learning led by a Code Coach. Code Coaching is available to adults as well. For more information, visit thecoderschool.com.

Center for Autism and Related Disorders (CARD)

12850 Jones Rd #105-106 Houston, TX 77070

CARD is a nationwide provider of applied behavior analysis (ABA) therapy for individuals with autism and other special needs. Their services include center-based treatment, remote clinical services, training, and more. For more information, visit centerforautism.com.
